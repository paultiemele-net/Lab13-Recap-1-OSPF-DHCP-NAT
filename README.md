# Lab13-Recap-1-OSPF-DHCP-NAT
A collection of Cisco networking labs demonstrating VLAN configuration, DHCP setup, dynamic and static NAT, OSPF routing, and multi-router communication scenarios. These labs are designed for learning and practicing advanced networking concepts using Cisco routers and switches in GNS3 or Packet Tracer.

Topics Covered:

VLAN configuration and trunking

DHCP server configuration per VLAN

Dynamic NAT and NAT pools

Static NAT

PAT (Port Address Translation)

OSPF routing across multiple areas

Multi-router communication

Connectivity verification using ping
